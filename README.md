## Getting Started

First, run the development server:

1. Install packages

```bash
npm install
# or
yarn
```

2. Run development server

```bash
npm run dev
# or
yarn dev
```

3. Open browser!

[http://localhost:3000](http://localhost:3000) 



## Stack

- Next.js
- Typescript
- React
- Mobx
- Emotion
- MSW
- Jest
- Prettier
- eslint
